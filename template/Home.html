What it does

Powered by AI & Machine Learning

🧠

LSTM Neural Network

A stacked LSTM model trained on mental health conversation patterns, using TensorFlow/Keras with layer normalization for stable training.

💬

Intent Classification

Multi-class NLP classification with tokenization, sequence padding, and label encoding to understand user input and route to best responses.

🔐

Secure Auth System

Full Django authentication with CSRF protection, password validation, and session management for user privacy and data security.

📊

Scikit-learn Preprocessing

Label encoding and Keras tokenizer pipelines with proper train/inference consistency — no data leakage between fit and transform steps.

⚡

REST API Backend

AJAX-powered chat with Django JsonResponse, async JS frontend, and real-time typing indicators for smooth user experience.

🚀

Production Ready

WhiteNoise static serving, environment-based config, proper model persistence, and Render/Railway deployment support out of the box.
